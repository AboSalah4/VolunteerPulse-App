### About the Project
**VolunteerPulse** is a responsive web application designed to help students with heavy course loads find "bite-sized" volunteer opportunities.

**Key Features:**
* **Time-Based Filtering:** Find tasks that fit in 15, 30, or 60-minute gaps.
* **One-Screen Flow:** Start a task in under 60 seconds (High Priority).
* **Impact Gamification:** Track "Pulse Points" and volunteer streaks.

### Tech Stack
* **Frontend:** React (Vite)
* **Backend:** Node.js & Express
* **Database:** MongoDB
* **Styling:** Tailwind CSS / CSS Modules
